# 📚 Online Bookstore SQL Project

## Project Overview

This project demonstrates an Online Bookstore database created using PostgreSQL.

It includes three tables:
- Books
- Customers
- Orders

The project analyzes sales, customers, inventory, and revenue using SQL queries.
